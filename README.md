# Thunderstorm Interactive Literary Archive

An AI-powered interactive literary archive for Cao Yu's *Thunderstorm*, built with Streamlit and the OpenAI API.

## Features

- Reading assistant for scene/act-based analysis
- Character archive with relationship and theme notes
- Character relationship graph with spoiler-free mode
- Conflict and theme dashboard
- Evidence search based on structured literary notes
- Upload/extend notes locally
- Export analysis history

## Tech Stack

- Python
- Streamlit
- OpenAI API
- JSON-based structured knowledge files
